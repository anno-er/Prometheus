前端面试中常考的算法题主要包括以下几类：

1. 数组相关
   两数之和：给定一个整数数组和一个目标值，找出数组中和为目标值的两个数
   三数之和：找出数组中所有和为 0 的不重复三元组
   移动零：将数组中的所有 0 移到数组末尾，同时保持非零元素的相对顺序
   移除元素：原地移除数组中所有等于给定值的元素
   合并两个有序数组：将两个有序数组合并为一个有序数组
2. 字符串相关
   反转字符串：原地反转字符数组
   最长公共前缀：查找字符串数组中的最长公共前缀
   有效的括号：判断括号字符串是否有效
   字符串转换整数：实现 atoi 函数功能
3. 链表相关
   反转链表：反转单链表
   合并两个有序链表：将两个升序链表合并为一个新的升序链表
   删除链表的倒数第 N 个节点：删除链表中倒数第 n 个节点
   环形链表检测：判断链表中是否有环
4. 排序和搜索
   二分查找：在有序数组中查找目标值
   搜索旋转排序数组：在旋转后的有序数组中搜索目标值
   快速排序：实现快速排序算法
   归并排序：实现归并排序算法
5. 动态规划
   爬楼梯：每次可以爬 1 或 2 个台阶，求有多少种不同的方法爬到楼顶
   最大子数组和：找出具有最大和的连续子数组
   买卖股票的最佳时机：设计算法获取最大利润
6. 栈和队列
   用栈实现队列：仅使用两个栈实现先入先出队列
   最小栈：设计一个支持 push、pop、top 操作并能在常数时间内检索到最小元素的栈
7. 树相关
   二叉树的层序遍历：按层遍历二叉树
   验证二叉搜索树：判断给定的二叉树是否为有效的二叉搜索树
   二叉树的最大深度：计算二叉树的最大深度
