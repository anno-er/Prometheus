# TypeScript 常见面试题

## 1. TypeScript 与 JavaScript 的区别是什么？

### 主要区别

- **类型系统**：TypeScript 是 JavaScript 的超集，添加了静态类型检查
- **编译时**：TypeScript 需要编译成 JavaScript 才能在浏览器运行
- **开发体验**：TypeScript 提供更好的 IDE 支持和代码提示
- **错误检测**：TypeScript 在编译时就能发现类型错误，JavaScript 在运行时才发现

### 示例

```typescript v-pre
// JavaScript
function add(a, b) {
  return a + b;
}

// TypeScript
function add(a: number, b: number): number {
  return a + b;
}
```

## 3. interface 和 type 的区别是什么？

### 相同点

- 都可以定义对象类型
- 都可以继承

### 不同点

- interface 可以声明合并，type 不可以
- type 可以定义联合类型、交叉类型、元组等，interface 不可以
- interface 更适合定义对象类型

```typescript v-pre
// interface 声明合并
interface User {
  name: string;
}

interface User {
  age: number;
}

const user: User = {
  name: 'TypeScript',
  age: 10,
};

// type 定义联合类型
type ID = string | number;

// type 定义元组
type Point = [number, number];
```

## 4. 什么是泛型？如何使用？

### 泛型概念

泛型是指在定义函数、接口或类时，不预先指定具体类型，而是在使用时再指定类型。

### 泛型函数

```typescript v-pre
function identity<T>(arg: T): T {
  return arg;
}

identity<string>('hello');
identity(123); // 类型推断
```

### 泛型接口

```typescript v-pre
interface Container<T> {
  value: T;
}

let container: Container<string> = {
  value: 'hello',
};
```

### 泛型类

```typescript v-pre
class GenericNumber<T> {
  zeroValue: T;
  add: (x: T, y: T) => T;
}

let myGenericNumber = new GenericNumber<number>();
myGenericNumber.zeroValue = 0;
myGenericNumber.add = function (x, y) {
  return x + y;
};
```

## 6. 什么是联合类型和交叉类型？

### 联合类型

联合类型表示一个值可以是几种类型之一。

```typescript v-pre
type StringOrNumber = string | number;

function padLeft(value: string, padding: StringOrNumber) {
  if (typeof padding === 'number') {
    return Array(padding + 1).join(' ') + value;
  }
  if (typeof padding === 'string') {
    return padding + value;
  }
  throw new Error(`Expected string or number, got '${padding}'.`);
}
```

### 交叉类型

交叉类型是将多个类型合并为一个类型。

```typescript v-pre
interface Person {
  name: string;
}

interface Loggable {
  log(): void;
}

type PersonLoggable = Person & Loggable;

const person: PersonLoggable = {
  name: 'TypeScript',
  log() {
    console.log(this.name);
  },
};
```

## 7. 什么是类型守卫？

### 类型守卫

类型守卫是一些表达式，它们会在运行时检查类型，并在某个作用域内保证类型。

### 常见类型守卫

```typescript v-pre
// typeof
function padLeft(value: string, padding: string | number) {
  if (typeof padding === 'number') {
    return Array(padding + 1).join(' ') + value;
  }
  if (typeof padding === 'string') {
    return padding + value;
  }
  throw new Error(`Expected string or number, got '${padding}'.`);
}

// instanceof
class Bird {
  fly() {
    console.log('flying...');
  }
}

class Fish {
  swim() {
    console.log('swimming...');
  }
}

function move(pet: Bird | Fish) {
  if (pet instanceof Bird) {
    pet.fly();
  } else {
    pet.swim();
  }
}

// in 操作符
interface Admin {
  name: string;
  privileges: string[];
}

interface Employee {
  name: string;
  startDate: Date;
}

type UnknownEmployee = Employee | Admin;

function printEmployeeInformation(emp: UnknownEmployee) {
  console.log('Name: ' + emp.name);
  if ('privileges' in emp) {
    console.log('Privileges: ' + emp.privileges);
  }
  if ('startDate' in emp) {
    console.log('Start Date: ' + emp.startDate);
  }
}
```

## 8. 什么是装饰器？

### 装饰器

装饰器是一种特殊类型的声明，它可以被附加到类声明、方法、属性或参数上，以修改类的行为。

### 类装饰器

```typescript v-pre
function sealed(constructor: Function) {
  Object.seal(constructor);
  Object.seal(constructor.prototype);
}

@sealed
class Greeter {
  greeting: string;
  constructor(message: string) {
    this.greeting = message;
  }
  greet() {
    return 'Hello, ' + this.greeting;
  }
}
```

### 方法装饰器

```typescript v-pre
function enumerable(value: boolean) {
  return function (
    target: any,
    propertyKey: string,
    descriptor: PropertyDescriptor
  ) {
    descriptor.enumerable = value;
  };
}

class Greeter {
  greeting: string;
  constructor(message: string) {
    this.greeting = message;
  }

  @enumerable(false)
  greet() {
    return 'Hello, ' + this.greeting;
  }
}
```

## 9. 什么是命名空间？

### 命名空间

命名空间用于组织代码，避免命名冲突。

```typescript v-pre
namespace Validation {
  export interface StringValidator {
    isAcceptable(s: string): boolean;
  }

  const lettersRegexp = /^[A-Za-z]+$/;
  const numberRegexp = /^[0-9]+$/;

  export class LettersOnlyValidator implements StringValidator {
    isAcceptable(s: string) {
      return lettersRegexp.test(s);
    }
  }

  export class ZipCodeValidator implements StringValidator {
    isAcceptable(s: string) {
      return s.length === 5 && numberRegexp.test(s);
    }
  }
}

let validators: { [s: string]: Validation.StringValidator } = {};
validators['ZIP code'] = new Validation.ZipCodeValidator();
validators['Letters only'] = new Validation.LettersOnlyValidator();
```

## 10. TypeScript 中的 never 类型是什么？

### never 类型

never 类型表示那些永不存在的值的类型。例如，never 类型是那些总是会抛出异常或根本就不会有返回值的函数表达式或箭头函数表达式的返回值类型。

```typescript v-pre
// 返回 never 的函数必须存在无法达到的终点
function error(message: string): never {
  throw new Error(message);
}

// 推断的返回值类型为 never
function fail() {
  return error('Something failed');
}

// 返回 never 的函数必须存在无法达到的终点
function infiniteLoop(): never {
  while (true) {}
}

// 利用 never 类型进行完整性检查
type Foo = string | number;

function controlFlowAnalysisWithNever(foo: Foo) {
  if (typeof foo === 'string') {
    // 这里 foo 被收窄为 string 类型
  } else if (typeof foo === 'number') {
    // 这里 foo 被收窄为 number 类型
  } else {
    // foo 在这里是 never
    const check: never = foo;
  }
}
```

## 11. any 和 unknown 的区别是什么？

### any 类型

any 类型是 TypeScript 中的"逃逸舱"，它会跳过类型检查。

```typescript v-pre
let anyValue: any = 'hello';

anyValue = 123;
anyValue = true;
anyValue = { name: 'TypeScript' };

anyValue.nonExistentMethod();
```

### unknown 类型

unknown 类型是类型安全的 any，使用前必须进行类型检查。

```typescript v-pre
let unknownValue: unknown = 'hello';

unknownValue = 123;
unknownValue = true;

// 错误：'unknownValue' is of type 'unknown'
// unknownValue.nonExistentMethod();

// 正确用法：先进行类型检查
if (typeof unknownValue === 'string') {
  console.log(unknownValue.toUpperCase());
}

// 使用类型断言
(unknownValue as string).toUpperCase();
```

### 主要区别

| 特性           | any       | unknown            |
| -------------- | --------- | ------------------ |
| 类型安全       | ❌ 不安全 | ✅ 安全            |
| 类型检查       | 跳过      | 强制检查           |
| 赋值给其他类型 | 允许      | 不允许（除非检查） |
| 访问属性       | 允许      | 不允许（除非检查） |

### 最佳实践

```typescript v-pre
function parseJSON(jsonString: string): unknown {
  return JSON.parse(jsonString);
}

const result = parseJSON('{"name": "TypeScript"}');

if (typeof result === 'object' && result !== null && 'name' in result) {
  console.log((result as { name: string }).name);
}
```

## 12. 什么是 keyof 关键字？

### keyof 操作符

keyof 是一个索引类型查询操作符，用于获取类型的所有键组成的联合类型。

### 基本用法

```typescript v-pre
interface Person {
  name: string;
  age: number;
  address: string;
}

type PersonKeys = keyof Person;

let key: PersonKeys = 'name';
key = 'age';
key = 'address';
// key = 'email'; // 错误
```

### 实际应用

```typescript v-pre
function getProperty<T, K extends keyof T>(obj: T, key: K): T[K] {
  return obj[key];
}

const person: Person = {
  name: 'TypeScript',
  age: 10,
  address: 'Microsoft',
};

const name = getProperty(person, 'name');
const age = getProperty(person, 'age');
```

### 与泛型结合

```typescript v-pre
function setProperty<T, K extends keyof T>(obj: T, key: K, value: T[K]): void {
  obj[key] = value;
}

setProperty(person, 'name', 'New Name');
setProperty(person, 'age', 11);
```

## 13. 什么是条件类型？

### 条件类型

条件类型根据条件判断来选择类型，类似于三元运算符。

### 基本语法

```typescript v-pre
T extends U ? X : Y
```

如果 T 可以赋值给 U，则类型为 X，否则为 Y。

### 示例

```typescript v-pre
type IsString<T> = T extends string ? true : false;

type A = IsString<string>; // true
type B = IsString<number>; // false
```

### 分布式条件类型

当条件类型作用于联合类型时，会自动分布到每个成员上。

```typescript v-pre
type ToArray<T> = T extends any ? T[] : never;

type StrArr = ToArray<string>; // string[]
type NumArr = ToArray<number>; // number[]
type StrOrNumArr = ToArray<string | number>; // string[] | number[]
```

### 实际应用

```typescript v-pre
type NonNullable<T> = T extends null | undefined ? never : T;

type A = NonNullable<string | null | undefined>; // string
type B = NonNullable<null>; // never
type C = NonNullable<undefined>; // never
```

## 14. 什么是映射类型？

### 映射类型

映射类型可以基于旧类型创建新类型，通过遍历键来转换类型。

### 基本语法

```typescript v-pre
type MappedType<T> = {
  [P in keyof T]: T[P];
};
```

### 常用映射类型

```typescript v-pre
interface Person {
  name: string;
  age: number;
}

type Readonly<T> = {
  readonly [P in keyof T]: T[P];
};

type Partial<T> = {
  [P in keyof T]?: T[P];
};

type Required<T> = {
  [P in keyof T]-?: T[P];
};

type Mutable<T> = {
  -readonly [P in keyof T]: T[P];
};

let readonlyPerson: Readonly<Person> = {
  name: 'TypeScript',
  age: 10,
};

let partialPerson: Partial<Person> = {
  name: 'TypeScript',
};

let requiredPerson: Required<{ name?: string }> = {
  name: 'TypeScript',
};
```

### 映射修饰符

- `+` 添加修饰符（可选、只读）
- `-` 移除修饰符

```typescript v-pre
interface ReadonlyPerson {
  readonly name: string;
  readonly age: number;
}

type MutablePerson = {
  -readonly [P in keyof ReadonlyPerson]: ReadonlyPerson[P];
};

let mutablePerson: MutablePerson = {
  name: 'TypeScript',
  age: 10,
};

mutablePerson.name = 'New Name';
```

## 15. 什么是 infer 关键字？

### infer 关键字

infer 用于在条件类型中推断类型，只能在 extends 子句中使用。

### 基本用法

```typescript v-pre
type ReturnType<T> = T extends (...args: any[]) => infer R ? R : any;

function add(a: number, b: number): number {
  return a + b;
}

type AddReturnType = ReturnType<typeof add>; // number
```

### 提取参数类型

```typescript v-pre
type Parameters<T> = T extends (...args: infer P) => any ? P : never;

function greet(name: string, age: number): string {
  return `Hello, ${name}! You are ${age} years old.`;
}

type GreetParams = Parameters<typeof greet>; // [string, number]
```

### 提取 Promise 返回类型

```typescript v-pre
type UnwrapPromise<T> = T extends Promise<infer U> ? U : T;

type A = UnwrapPromise<Promise<string>>; // string
type B = UnwrapPromise<string>; // string
```

### 提取数组元素类型

```typescript v-pre
type ElementType<T> = T extends (infer E)[] ? E : T;

type A = ElementType<string[]>; // string
type B = ElementType<number[]>; // number
type C = ElementType<string>; // string
```

### 提取函数第一个参数类型

```typescript v-pre
type FirstParam<T> = T extends (first: infer F, ...rest: any[]) => any
  ? F
  : never;

function example(first: string, second: number, third: boolean): void {}

type First = FirstParam<typeof example>; // string
```

## 16. TypeScript 内置工具类型有哪些？

### Partial&lt;T&gt;

将所有属性变为可选。

```typescript v-pre
interface User {
  name: string;
  age: number;
  email: string;
}

type PartialUser = Partial<User>;

const user: PartialUser = {
  name: 'TypeScript',
};

type MyPartial<T> = {
  [P in keyof T]?: T[P];
};
```

### Required&lt;T&gt;

将所有属性变为必选。

```typescript v-pre
interface User {
  name?: string;
  age?: number;
}

type RequiredUser = Required<User>;

const user: RequiredUser = {
  name: 'TypeScript',
  age: 10,
};

type MyRequired<T> = {
  [P in keyof T]: T[P];
};
```

### Readonly&lt;T&gt;

将所有属性变为只读。

```typescript v-pre
interface User {
  name: string;
  age: number;
}

type ReadonlyUser = Readonly<User>;

const user: ReadonlyUser = {
  name: 'TypeScript',
  age: 10,
};

// user.name = 'New Name'; // 错误

type MyReadonly<T> = {
  readonly [P in keyof T]: T[P];
};
```

### Pick&lt;T, K&gt;

选择部分属性。

```typescript v-pre
interface User {
  name: string;
  age: number;
  email: string;
  address: string;
}

type UserNameAndAge = Pick<User, 'name' | 'age'>;

const user: UserNameAndAge = {
  name: 'TypeScript',
  age: 10,
};

type MyPick<T, U extends keyof T> = {
  [P in U]: T[P];
};
```

### Omit&lt;T, K&gt;

排除部分属性。

```typescript v-pre
interface User {
  name: string;
  age: number;
  email: string;
  address: string;
}

type UserWithoutEmail = Omit<User, 'email' | 'address'>;

const user: UserWithoutEmail = {
  name: 'TypeScript',
  age: 10,
};

type MyOmit<T, U extends keyof T> = {
  [P in Exclude<keyof T, U>]: T[P];
};
```

### Record&lt;K, T&gt;

创建键值对类型。

```typescript v-pre
type UserRecord = Record<string, { name: string; age: number }>;

const users: UserRecord = {
  user1: { name: 'Alice', age: 25 },
  user2: { name: 'Bob', age: 30 },
};

type MyRecord<K extends keyof any, T> = {
  [P in K]: T;
};
// extends keyof any: 约束 K 为任意键类型 string | number | symbol
```

### Extract&lt;T, U&gt;

提取可以赋值给 U 的类型。

```typescript v-pre
type T = string | number | boolean;
type StringOrNumber = Extract<T, string | number>; // string | number

type MyExtract<T, U> = T extends U ? T : never;
// T extends U: 当 T 是联合类型时，条件类型会自动分发，逐个判断每个成员是否符合 U
```

### Exclude&lt;T, U&gt;

排除可以赋值给 U 的类型。

```typescript v-pre
type T = string | number | boolean;
type BooleanOnly = Exclude<T, string | number>; // boolean

type MyExclude<T, U> = T extends U ? never : T;
```

### NonNullable&lt;T&gt;

排除 null 和 undefined。

```typescript v-pre
type T = string | null | undefined;
type NonNull = NonNullable<T>; // string

type MyNonNullable<T> = T extends null | undefined ? never : T;
```

### ReturnType&lt;T&gt;

获取函数返回类型。

```typescript v-pre
function createUser() {
  return {
    name: 'TypeScript',
    age: 10,
  };
}

type User = ReturnType<typeof createUser>;
// { name: string; age: number; }

type MyReturnType<T> = T extends (...args: any) => infer R ? R : never;
```

### Parameters&lt;T&gt;

获取函数参数类型。

```typescript v-pre
function greet(name: string, age: number): string {
  return `Hello, ${name}!`;
}

type Params = Parameters<typeof greet>; // [string, number]

type MyParameters<T> = T extends (...args: infer R) => any ? R : never;
```

### InstanceType&lt;T&gt;

获取构造函数实例类型。

```typescript v-pre
class Person {
  constructor(public name: string, public age: number) {}
}

type PersonInstance = InstanceType<typeof Person>;

const person: PersonInstance = new Person('TypeScript', 10);

type MyInstanceType<T> = T extends new (...args: any) => infer R ? R : never;
```

## 17. 什么是函数重载？

### 函数重载

函数重载允许为同一个函数提供多个类型定义，根据参数类型决定返回类型。

### 基本用法

```typescript v-pre
function add(a: number, b: number): number;
function add(a: string, b: string): string;
function add(a: any, b: any): any {
  return a + b;
}

const numResult = add(1, 2); // number
const strResult = add('Hello, ', 'TypeScript'); // string
```

### 实际应用

```typescript v-pre
interface User {
  name: string;
  age: number;
}

function createUser(name: string): User;
function createUser(name: string, age: number): User;
function createUser(name: string, age?: number): User {
  return {
    name,
    age: age ?? 0,
  };
}

const user1 = createUser('TypeScript');
const user2 = createUser('TypeScript', 10);
```

### 事件处理示例

```typescript v-pre
type EventType = 'click' | 'hover' | 'scroll';

function on(event: 'click', handler: (e: MouseEvent) => void): void;
function on(event: 'hover', handler: (e: MouseEvent) => void): void;
function on(event: 'scroll', handler: (e: Event) => void): void;
function on(event: EventType, handler: (e: Event) => void): void {
  console.log(`Registered ${event} handler`);
}
```

## 18. 什么是可选链和空值合并？

### 可选链操作符 ?.

可选链操作符用于安全地访问深层嵌套的对象属性。

```typescript v-pre
interface User {
  name: string;
  address?: {
    city?: string;
    street?: string;
  };
}

const user: User = {
  name: 'TypeScript',
};

const city = user.address?.city; // undefined
const street = user.address?.street?.toUpperCase(); // undefined
```

### 空值合并操作符 ??

空值合并操作符用于提供默认值，仅在值为 null 或 undefined 时使用默认值。

```typescript v-pre
const value: string | null = null;
const result = value ?? 'default'; // 'default'

const value2: string | undefined = undefined;
const result2 = value2 ?? 'default'; // 'default'

const value3: string = '';
const result3 = value3 ?? 'default'; // ''

const value4: number = 0;
const result4 = value4 ?? 10; // 0
```

### 与 || 的区别

```typescript v-pre
const value = '';
const result1 = value || 'default'; // 'default' (空字符串是 falsy)
const result2 = value ?? 'default'; // '' (空字符串不是 null/undefined)

const value2 = 0;
const result3 = value2 || 10; // 10 (0 是 falsy)
const result4 = value2 ?? 10; // 0 (0 不是 null/undefined)

const value3 = false;
const result5 = value3 || true; // true (false 是 falsy)
const result6 = value3 ?? true; // false (false 不是 null/undefined)
```

### 结合使用

```typescript v-pre
interface Config {
  database?: {
    host?: string;
    port?: number;
  };
}

function getDatabaseUrl(config: Config): string {
  const host = config.database?.host ?? 'localhost';
  const port = config.database?.port ?? 3306;
  return `${host}:${port}`;
}
```

## 19. 什么是类型推断？

### 类型推断

TypeScript 会在没有明确指定类型时自动推断类型。

### 变量推断

```typescript v-pre
let x = 10; // number
let y = 'hello'; // string
let z = [1, 2, 3]; // number[]
let obj = { name: 'TypeScript', age: 10 }; // { name: string; age: number; }

// x = 'string'; // 错误
```

### 函数返回值推断

```typescript v-pre
function add(a: number, b: number) {
  return a + b; // 返回类型推断为 number
}

function greet(name: string) {
  return `Hello, ${name}!`; // 返回类型推断为 string
}
```

## 23. 什么是协变和逆变？

### 协变（Covariance）

子类型可以赋值给父类型。

```typescript v-pre
interface Animal {
  name: string;
}

interface Dog extends Animal {
  breed: string;
}

let animals: Animal[] = [];
let dogs: Dog[] = [];

// animals = dogs; // 在严格模式下会报错
```

### 逆变（Contravariance）

父类型可以赋值给子类型（在函数参数中）。

```typescript v-pre
type AnimalHandler = (animal: Animal) => void;
type DogHandler = (dog: Dog) => void;

let handleAnimal: AnimalHandler = (animal) => {
  console.log(animal.name);
};

let handleDog: DogHandler = handleAnimal; // 正确
```

### 不变（Invariance）

类型必须完全匹配。

```typescript v-pre
interface Box<T> {
  value: T;
  setValue(value: T): void;
}

let stringBox: Box<string> = {
  value: 'hello',
  setValue(value: string) {
    this.value = value;
  },
};

// let objectBox: Box<object> = stringBox; // 错误
```

## 24. 什么是 this 类型？

### this 类型

this 类型表示某个类或接口的子类型，用于实现链式调用。

### 基本用法

```typescript v-pre
class Calculator {
  protected value: number = 0;

  add(n: number): this {
    this.value += n;
    return this;
  }

  subtract(n: number): this {
    this.value -= n;
    return this;
  }

  multiply(n: number): this {
    this.value *= n;
    return this;
  }

  getValue(): number {
    return this.value;
  }
}

const calc = new Calculator();
const result = calc.add(10).subtract(5).multiply(2).getValue(); // 10
```

### 继承中的 this 类型

```typescript v-pre
class ScientificCalculator extends Calculator {
  power(n: number): this {
    this.value = Math.pow(this.value, n);
    return this;
  }
}

const sciCalc = new ScientificCalculator();
const result = sciCalc.add(10).power(2).getValue(); // 100
```

### 在接口中使用

```typescript v-pre
interface MyObject {
  name: string;
  setName(name: string): this;
  getName(): string;
}

const obj: MyObject = {
  name: '',
  setName(name: string) {
    this.name = name;
    return this;
  },
  getName() {
    return this.name;
  },
};

obj.setName('TypeScript').getName(); // 'TypeScript'
```

## 26. 什么是可辨识联合？

### 可辨识联合

可辨识联合是一种使用公共属性来区分联合类型成员的模式。

### 基本用法

```typescript v-pre
interface Circle {
  kind: 'circle';
  radius: number;
}

interface Rectangle {
  kind: 'rectangle';
  width: number;
  height: number;
}

interface Triangle {
  kind: 'triangle';
  base: number;
  height: number;
}

type Shape = Circle | Rectangle | Triangle;

function getArea(shape: Shape): number {
  switch (shape.kind) {
    case 'circle':
      return Math.PI * shape.radius ** 2;
    case 'rectangle':
      return shape.width * shape.height;
    case 'triangle':
      return (shape.base * shape.height) / 2;
  }
}
```

### 穷尽检查

```typescript v-pre
function getArea(shape: Shape): number {
  switch (shape.kind) {
    case 'circle':
      return Math.PI * shape.radius ** 2;
    case 'rectangle':
      return shape.width * shape.height;
    case 'triangle':
      return (shape.base * shape.height) / 2;
    default:
      const _exhaustiveCheck: never = shape;
      throw new Error(`Unknown shape: ${_exhaustiveCheck}`);
  }
}
```

### Redux Action 示例

```typescript v-pre
interface AddTodoAction {
  type: 'ADD_TODO';
  payload: {
    id: number;
    text: string;
  };
}

interface RemoveTodoAction {
  type: 'REMOVE_TODO';
  payload: {
    id: number;
  };
}

interface ToggleTodoAction {
  type: 'TOGGLE_TODO';
  payload: {
    id: number;
  };
}

type TodoAction = AddTodoAction | RemoveTodoAction | ToggleTodoAction;

function todoReducer(state: Todo[], action: TodoAction): Todo[] {
  switch (action.type) {
    case 'ADD_TODO':
      return [...state, { ...action.payload, completed: false }];
    case 'REMOVE_TODO':
      return state.filter((todo) => todo.id !== action.payload.id);
    case 'TOGGLE_TODO':
      return state.map((todo) =>
        todo.id === action.payload.id
          ? { ...todo, completed: !todo.completed }
          : todo
      );
  }
}
```

## 29. 什么是类型收窄？

### 类型收窄

类型收窄是指通过类型守卫将类型从较宽泛的类型收窄到更具体的类型。

### typeof 收窄

```typescript v-pre
function printAll(strs: string | string[] | null) {
  if (typeof strs === 'object') {
    for (const s of strs!) {
      console.log(s);
    }
  } else if (typeof strs === 'string') {
    console.log(strs);
  } else {
    // do nothing
  }
}
```

### 真值收窄

```typescript v-pre
function getUsersOnlineMessage(numUsersOnline: number) {
  if (numUsersOnline) {
    return `There are ${numUsersOnline} online now!`;
  }
  return "Nobody's here. :(";
}
```

### 等值收窄

```typescript v-pre
function example(x: string | number, y: string | boolean) {
  if (x === y) {
    x.toUpperCase(); // 正确，x 是 string
    y.toUpperCase(); // 正确，y 是 string
  } else {
    console.log(x);
    console.log(y);
  }
}
```

### instanceof 收窄

```typescript v-pre
function logValue(x: Date | string) {
  if (x instanceof Date) {
    console.log(x.toUTCString());
  } else {
    console.log(x.toUpperCase());
  }
}
```

### in 操作符收窄

```typescript v-pre
interface Fish {
  swim: () => void;
}

interface Bird {
  fly: () => void;
}

function move(pet: Fish | Bird) {
  if ('swim' in pet) {
    pet.swim();
  } else {
    pet.fly();
  }
}
```

### 赋值收窄

```typescript v-pre
let x = Math.random() < 0.5 ? 10 : 'hello world!';

x = 29; // x 变为 number

x = 'good bye!'; // x 变为 string
```

### 类型谓词

```typescript v-pre
interface Fish {
  name: string;
  swim: () => void;
}

interface Bird {
  name: string;
  fly: () => void;
}

function isFish(pet: Fish | Bird): pet is Fish {
  return (pet as Fish).swim !== undefined;
}

function move(pet: Fish | Bird) {
  if (isFish(pet)) {
    pet.swim();
  } else {
    pet.fly();
  }
}
```

### 断言函数

```typescript v-pre
function assertIsString(value: unknown): asserts value is string {
  if (typeof value !== 'string') {
    throw new Error('Value is not a string');
  }
}

function process(value: unknown) {
  assertIsString(value);
  console.log(value.toUpperCase()); // 正确，value 是 string
}
```
