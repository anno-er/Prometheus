# 017_CSS 属性继承

CSS 中的属性分为可继承和不可继承两类。可继承属性会从父元素传递给子元素，而不可继承属性则只应用于定义它们的元素。一般来说，可继承属性主要涉及文本和字体相关的属性，以及一些布局相关属性。

## 1. 可继承属性

以下属性可以从父元素继承到子元素：

- `color`
- `font-family`
- `font-size`
- `font-style`
- `font-variant`
- `font-weight`
- `letter-spacing`
- `line-height`
- `text-align`
- `text-indent`
- `text-transform`
- `visibility`
- `white-space`
- `word-spacing`

## 2. 不可继承属性

以下属性不会从父元素继承到子元素：

- `background`
- `border`
- `display`
- `height`
- `margin`
- `padding`
- `width`
- `position`
- `top`, `right`, `bottom`, `left`
- `z-index`
- `float`
- `clear`

## 3. 继承关键字

`inherit` 关键字用于指定一个属性从父元素继承

`initial` 关键字用于将一个属性重置为默认值

`unset` 对于可继承属性,行为与`inherit`相同;对于不可继承属性,行为与`initial`相同
