# 016_CSS 选择器与优先级

## CSS 选择器与权重

| 等级 | 来源 | 表示方式 | 举例 | 权重值（a,b,c,d） |
|------|------|----------|------|-------------------|
| 1 | **内联样式** (Inline Style) | `style="..."` | `<div style="color:red;">` | (1, 0, 0, 0) → 最高优先级 |
| 2 | **ID 选择器** | `#id` | `#header` | (0, 1, 0, 0) |
| 3 | **类选择器 / 伪类选择器 / 属性选择器** | `.class`, `:hover`, `[type="text"]` | `.btn`, `:focus`, `[name=user]` | (0, 0, 1, 0) |
| 4 | **元素选择器 / 伪元素选择器** | `div`, `p`, `::before` | `div`, `::first-line` | (0, 0, 0, 1) |
| 5 | **通配符选择器 / 继承的样式 / 浏览器默认样式** | `*`, `div p`（后代选择器本身不增加权重，只看子选择器） | `*`, `div p` | (0, 0, 0, 0) → **最低** |

## 选择器权重计算

```css
div#main.content > ul li:first-child
```

拆解：

div → (0,0,0,1)
#main → (0,1,0,0)
.content → (0,0,1,0)

> → （无权重）
> ul → (0,0,0,1)
> li → (0,0,0,1)
> :first-child → (0,0,1,0)
> 总权重 = (0, 1, 3, 3)

对比规则： 从高到低，按位对比，底层级相加无法突破高层级，即使类选择器相加超过一百万优先级也低于 id 选择器
