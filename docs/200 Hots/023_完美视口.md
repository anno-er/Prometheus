# 完美视口

## 什么是视口(Viewport)

视口是浏览器中用来显示网页的区域，它不包括浏览器的 UI 界面（如地址栏、书签栏等）。在移动端设备上，视口的概念尤为重要，因为它直接影响到网页在不同设备上的显示效果。

## CSS 像素与设备像素

- **CSS 像素**：Web 页面中使用的逻辑像素单位
- **设备像素**：屏幕物理像素点的数量
- **设备像素比(DPR)**：设备像素与 CSS 像素的比例关系

## 移动端视口类型

### 1. 布局视口(Layout Viewport)

- 浏览器默认创建的一个虚拟视口，宽度通常为 980px
- 页面按照布局视口进行渲染和布局

### 2. 视觉视口(Visual Viewport)

- 用户当前实际看到的区域
- 用户缩放时视觉视口大小会发生变化

### 3. 理想视口(Ideal Viewport)

- 针对特定设备的最佳视口尺寸
- 与设备屏幕宽度一致

## 完美视口的实现

通过 meta viewport 标签设置理想视口：

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0" />
```

- width=device-width：设置视口宽度等于设备宽度
- initial-scale=1.0：初始缩放比例
- maximum-scale、minimum-scale：最大最小缩放比例
- user-scalable：是否允许用户缩放
