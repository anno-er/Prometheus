# HTML5 前端面试知识点

## HTML5 新特性

### 语义化标签

- `<header>`: 页面或章节的头部
- `<nav>`: 导航链接区域
- `<main>`: 页面主要内容
- `<article>`: 独立的文章内容
- `<section>`: 文档中的节
- `<aside>`: 侧边栏内容
- `<footer>`: 页面或章节的底部
- `<figure>` 和 `<figcaption>`: 媒体内容及其标题

### 多媒体元素

- `<audio>`: 音频播放元素
- `<video>`: 视频播放元素
- `<canvas>`: 用于绘制图形的画布元素

### 表单增强

- 新的输入类型：`email`, `url`, `number`, `date`, `range` 等
- 新的表单属性：`placeholder`, `required`, `autofocus` 等
- 表单验证 API

### Web 存储

- localStorage: 持久化存储数据
- sessionStorage: 会话期间存储数据
- 与 cookie 的区别及优势

### Web Workers

- 后台运行 JavaScript，不阻塞 UI 线程
- 主线程与 worker 线程通信机制

### WebSocket

- 全双工通信协议
- 与 HTTP 轮询的比较
- 实时通信应用场景

## 常见面试题

### 1. HTML5 与 HTML4 的区别

- 引入了新的语义化标签
- 增强了表单控件和属性
- 提供了本地存储解决方案
- 支持音频和视频元素
- 提供 Canvas 绘图功能
- 支持 Web Workers 和 WebSocket

### 2. localStorage、sessionStorage 和 cookie 的区别

| 特性      | localStorage   | sessionStorage   | cookie         |
| --------- | -------------- | ---------------- | -------------- |
| 生命周期  | 持久存储       | 会话结束清除     | 可设置过期时间 |
| 存储大小  | 5-10MB         | 5-10MB           | 4KB            |
| HTTP 请求 | 不发送到服务器 | 不发送到服务器   | 每次请求携带   |
| 作用域    | 同源共享       | 同源且同窗口共享 | 可设置作用域   |

### 3. Canvas 与 SVG 的区别

- Canvas: 基于像素的位图，适合游戏和复杂图像处理
- SVG: 基于矢量的图形，适合图标和简单图形，可缩放不失真

### 4. 如何实现 HTML5 离线存储

- 创建 manifest 文件
- 在 html 标签中引用 manifest
- 监听 applicationCache 事件

### 5. HTML5 新增的表单属性有哪些

- `placeholder`: 输入框提示信息
- `required`: 必填字段
- `autofocus`: 自动聚焦
- `pattern`: 正则表达式验证
- `min`/`max`: 数值范围限制

## 最佳实践

### 1. 语义化的重要性

- 提升 SEO 效果
- 增强可访问性
- 便于维护和团队协作
- 有利于浏览器解析和搜索引擎抓取

### 2. 移动端适配方案

- viewport 设置
- 响应式设计
- touch 事件处理
- 移动端调试技巧

### 3. 性能优化建议

- 减少 DOM 节点数量
- 合理使用 Web Workers 避免阻塞
- 图片懒加载实现
- 本地缓存策略选择
