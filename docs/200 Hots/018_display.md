# CSS Display 属性

display 属性是用于定义元素的显示,即元素应如何在页面上布局和显示。每种取值都影响元素的渲染方式和布局特性。以下是常见的 display 属性值及其作用。

| 属性值 | 是否独占一行 | 是否同行显示 | 是否可设置宽高 | 是否可设置margin/padding | 说明 |
|--------|-------------|-------------|---------------|-------------------------|------|
| `block` | ✅ 是 | ❌ 否 | ✅ 是 | ✅ 可以 | 默认块元素，如 `<div>` |
| `inline` | ❌ 否 | ✅ 是 | ❌ 否 | ❌ 不可 | 默认行内元素，如 `<span>` |
| `inline-block` | ❌（但可控制） | ✅（同行） | ❌ 否 | ✅ 可以 | 按钮、图标等需要内联又想设宽高的元素 |
| `flex` | ✅ 是 | ❌ 否 | ✅ 是 | ✅ 可以 | 弹性布局（现代布局首选之一） |
| `grid` | ✅ 是 | ❌ 否 | ✅ 是 | ✅ 可以 | 网格布局（二维布局方案） |
| `none` | ❌ 否 | ❌ 否 | ❌ 否 | ❌ 不可 | **完全隐藏且不占空间**（常用） |
| `table` | ✅ 是 | ❌ 否 | ✅ 是 | ✅ 可以 | 模拟表格布局（很少直接用） |
