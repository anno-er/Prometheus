# ES6 新特性

ES6 (ECMAScript 2015) 是 JavaScript 语言的一次重大更新，引入了许多新特性，大大提升了开发效率和代码可读性。

## 1. let 和 const

### let

具有块级作用域，用来声明变量可重新赋值。

```javascript
function example() {
  if (true) {
    let x = 10;
    console.log(x); // 10
  }
  // console.log(x); // ReferenceError: x is not defined
}

let y = 5;
y = 15; // 可以重新赋值
```

### const

具有块级作用域，用来声明常量不可再次赋值。

```javascript
const PI = 3.14159;
// PI = 3.14; // TypeError: Assignment to constant variable

const person = { name: "张三" };
person.name = "李四"; // 可以修改对象属性
// person = {}; // TypeError: Assignment to constant variable
```

## 2. 箭头函数

新的函数声明方式，语法简洁，自动绑定 this。

```javascript
// 传统函数
function add(a, b) {
  return a + b;
}

// 箭头函数
const add = (a, b) => a + b;

// 单个参数可以省略括号
const square = (x) => x * x;

// 多行语句需要大括号
const greet = (name) => {
  const message = `你好, ${name}!`;
  return message;
};
```

## 3. 模版字符串

字符串插值功能，可定义多行字符串。

```javascript
const name = "张三";
const age = 25;

// 字符串插值
const message = `你好，我是${name}，今年${age}岁。`;

// 多行字符串
const html = `
  <div>
    <h1>${name}</h1>
    <p>年龄: ${age}</p>
  </div>
`;

// 表达式计算
const result = `1 + 2 = ${1 + 2}`;
```

## 4. 解构赋值

是一种 JavaScript 表达式，它允许从数组或对象中提取属性或值，并将这些值赋给其他变量。

```javascript
// 数组解构
const [a, b, c] = [1, 2, 3];
console.log(a, b, c); // 1 2 3

// 对象解构
const person = { name: "张三", age: 25, city: "北京" };
const { name, age } = person;
console.log(name, age); // 张三 25

// 重命名
const { name: personName } = person;
console.log(personName); // 张三

// 默认值
const { country = "中国" } = person;
console.log(country); // 中国
```

## 5. 默认参数

函数参数可设置默认值。

```javascript
function greet(name = "访客", greeting = "你好") {
  return `${greeting}, ${name}!`;
}

console.log(greet()); // 你好, 访客!
console.log(greet("张三")); // 你好, 张三!
console.log(greet("张三", "欢迎")); // 欢迎, 张三!

// 默认参数可以是表达式
function calculateArea(width = 10, height = width * 2) {
  return width * height;
}
```

## 6. 扩展运算符

可以将数组展开为逗号分隔的参数序列，或者合并多个对象或数组。

```javascript
// 数组展开
const numbers = [1, 2, 3];
const moreNumbers = [...numbers, 4, 5, 6];
console.log(moreNumbers); // [1, 2, 3, 4, 5, 6]

// 函数调用
const nums = [1, 2, 3];
console.log(Math.max(...nums)); // 3

// 对象合并
const obj1 = { a: 1, b: 2 };
const obj2 = { c: 3, d: 4 };
const merged = { ...obj1, ...obj2 };
console.log(merged); // { a: 1, b: 2, c: 3, d: 4 }

// 数组复制
const original = [1, 2, 3];
const copy = [...original];
```

## 7. 类与模块

通过 class 关键字定义类，使用 import 和 export 来导入和导出模块。

### 类

```javascript
class Person {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }

  greet() {
    return `你好，我是${this.name}，今年${this.age}岁。`;
  }

  static species() {
    return "人类";
  }
}

class Student extends Person {
  constructor(name, age, grade) {
    super(name, age);
    this.grade = grade;
  }

  study() {
    return `${this.name}正在学习${this.grade}年级的课程。`;
  }
}
```

### 模块

```javascript
// math.js - 导出模块
export const PI = 3.14159;

export function add(a, b) {
  return a + b;
}

export default class Calculator {
  multiply(a, b) {
    return a * b;
  }
}

// main.js - 导入模块
import Calculator, { PI, add } from "./math.js";

const calc = new Calculator();
console.log(add(2, 3)); // 5
console.log(calc.multiply(4, 5)); // 20
```

## 8. Promise

用于处理异步操作，提供了更优雅的异步编程解决方案。

```javascript
// 创建 Promise
const fetchData = new Promise((resolve, reject) => {
  setTimeout(() => {
    const data = { id: 1, name: "张三" };
    // resolve(data); // 成功
    reject(new Error("获取数据失败")); // 失败
  }, 1000);
});

// 使用 Promise
fetchData
  .then((data) => {
    console.log("成功:", data);
  })
  .catch((error) => {
    console.log("失败:", error.message);
  });

// Promise.all - 并行执行多个异步操作
const promise1 = Promise.resolve(1);
const promise2 = Promise.resolve(2);
const promise3 = Promise.resolve(3);

Promise.all([promise1, promise2, promise3]).then((results) => {
  console.log(results); // [1, 2, 3]
});

// async/await - 基于 Promise 的语法糖
async function getData() {
  try {
    const result = await fetchData;
    console.log(result);
  } catch (error) {
    console.log(error.message);
  }
}
```

## 9. Symbol 和迭代器

提供了一种新的原始数据类型和自定义迭代行为的方式。

### Symbol

```javascript
// 创建 Symbol
const sym1 = Symbol("description");
const sym2 = Symbol("description");

console.log(sym1 === sym2); // false - 每个 Symbol 都是唯一的

// 用作对象属性键
const obj = {
  [sym1]: "私有属性",
  public: "公开属性",
};

console.log(Object.keys(obj)); // ['public'] - Symbol 属性不会被枚举
```

### 迭代器

```javascript
// 自定义迭代器
const myIterable = {
  data: [1, 2, 3],
  [Symbol.iterator]() {
    let index = 0;
    const data = this.data;

    return {
      next() {
        if (index < data.length) {
          return { value: data[index++], done: false };
        } else {
          return { done: true };
        }
      },
    };
  },
};

// 使用 for...of 循环
for (const value of myIterable) {
  console.log(value); // 1, 2, 3
}

// 生成器函数 - 简化迭代器创建
function* generateSequence() {
  yield 1;
  yield 2;
  yield 3;
}

const generator = generateSequence();
console.log(generator.next()); // { value: 1, done: false }
```

## 10. 新的数据结构

### Map

```javascript
// 创建 Map
const map = new Map();

// 设置键值对
map.set("name", "张三");
map.set("age", 25);
map.set({ key: "object" }, "对象作为键");

// 获取值
console.log(map.get("name")); // 张三

// 检查键是否存在
console.log(map.has("age")); // true

// 遍历
for (const [key, value] of map) {
  console.log(key, value);
}

// Map vs Object
// Map 可以使用任意类型的键，保持插入顺序，有 size 属性
```

### Set

```javascript
// 创建 Set
const set = new Set();

// 添加元素
set.add(1);
set.add(2);
set.add(1); // 重复值会被忽略

console.log(set); // Set { 1, 2 }

// 检查元素是否存在
console.log(set.has(2)); // true

// 删除元素
set.delete(1);

// 遍历
for (const value of set) {
  console.log(value);
}

// 数组去重
const numbers = [1, 2, 3, 2, 4, 1, 5];
const uniqueNumbers = [...new Set(numbers)];
console.log(uniqueNumbers); // [1, 2, 3, 4, 5]
```

## 11. 其他特性

### 对象属性简写

```javascript
const name = "张三";
const age = 25;

// ES5
const person1 = {
  name: name,
  age: age,
  sayHello: function () {
    return "你好";
  },
};

// ES6 - 属性简写
const person2 = {
  name, // 等同于 name: name
  age, // 等同于 age: age
  sayHello() {
    // 方法简写
    return "你好";
  },
};
```

### 计算属性名

```javascript
const propName = "dynamic";

const obj = {
  [propName]: "动态属性",
  [`${propName}Key`]: "组合属性名",
};

console.log(obj); // { dynamic: '动态属性', dynamicKey: '组合属性名' }
```

### for...of 循环

```javascript
const array = [1, 2, 3];
const string = "Hello";

// 遍历数组
for (const value of array) {
  console.log(value); // 1, 2, 3
}

// 遍历字符串
for (const char of string) {
  console.log(char); // H, e, l, l, o
}
```

## 总结

ES6 的这些新特性大大提升了 JavaScript 的开发体验：

- **let/const** 提供了更好的变量作用域控制
- **箭头函数** 简化了函数定义并解决了 this 绑定问题
- **模版字符串** 让字符串处理更加直观
- **解构赋值** 简化了数据提取
- **默认参数** 让函数定义更加灵活
- **扩展运算符** 提供了更优雅的数据操作方式
- **类与模块** 引入了现代化的面向对象和模块化编程
- **Promise** 改善了异步编程体验
- **Symbol 和迭代器** 提供了新的数据类型和迭代机制
- **Map 和 Set** 丰富了数据结构选择
- **各种语法糖** 提升了代码的可读性和开发效率

这些特性共同构成了现代 JavaScript 开发的基础，是每个 JavaScript 开发者都应该掌握的核心知识。
